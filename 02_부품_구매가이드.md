# 부품 구매 가이드 — 총 ~$18

## 0. 왜 이렇게 싼가 / 측정 장비 개념

센서 자체(스피커+마이크+스텐실)는 ~$10다. 유일한 진짜 비용은 **40kHz를 받을 측정 장비**:
- 40kHz는 사람 귀·일반 마이크·PC 사운드카드(최대 48kHz)가 **못 잡는다**. Nyquist상 **>80kHz 샘플링**이 필요.
- 보통 ADALM2000(~$287) 같은 실험실 장비를 쓰지만, **보유한 ESP32-S3가 같은 일을 $0에** 한다:
  - **송신:** LEDC PWM으로 40kHz 구형파 출력 (피에조가 협대역 필터링해 사인파화 → DAC 불필요)
  - **수신:** ADC 연속(DMA) 모드로 마이크 신호 샘플링 → USB 시리얼로 PC 전송
- **"펌웨어"(ESP32에 올리는 프로그램)는 이 저장소 `bench/esp32_firmware/`에 제공된다.** 사용자는 Arduino IDE로 업로드만.

### TCT40-16 페어가 가장 저렴 (한국)
송신 `T` + 수신 `R` 한 쌍이 알리/디바이스마트에서 ~$1~3. 16mm 직경, 40kHz±1kHz. 1·2번을 이걸로 한 번에 해결. 여분 위해 10T+10R 세트(~$8)도 좋음.

## 2. 배선 개요 (브레드보드)

```
ESP32-S3 GPIO(PWM) ──> [SN74LVC2G04 버퍼] ──> [40kHz 스피커]   (송신)
                                                  ↓ ))) 스텐실
40kHz 마이크 ──> [OPA344 프리앰프 ~40dB + 밴드패스] ──> ESP32-S3 ADC핀  (수신)
                   └ DC 바이어스 +1.65V (ADC 0~3.3V 중앙)
ESP32-S3 ──USB──> PC (시리얼로 샘플 스트림)
```
- 프리앰프 출력에 **+1.65V DC 바이어스**(저항 분압 + 캐패시터) — ADC가 음수 못 읽으므로 중앙 정렬.
- 스피커와 마이크 사이에 **흡음 배플** — 직접 누설(crosstalk) 차단이 근거리 성능의 핵심.
- 상세 회로·핀맵은 [05_벤치_및_실측H_가이드](05_벤치_및_실측H_가이드.md).

## 3. 한국 구매 메모

- **디바이스마트(devicemart.co.kr) / 엘레파츠(eleparts.co.kr):** OPA344, MCP6002, SN74LVC2G04, 트랜스듀서, 패시브 — 국내 당일~2일.
- **알리익스프레스:** TCT40-16 페어, 트랜스듀서 세트 — 최저가, 배송 1~3주.
- **DigiKey/Mouser 코리아:** Murata 정품 — 정확하나 배송비·관세(15~20%) 가산.
- 직구 가격은 USD 기준, 한국 도착가 = ×환율 + 배송 + 관세.

## 4. 주의 / 폴백

- ⚠️ **ESP32-S3 ADC 연속 100~200kHz 샘플링은 G1에서 반드시 실측 검증.** 기존 `C_SPiDR_재현_구매리스트.md`도 [CRITICAL]로 미검증 표기. 연속 DMA에서 지터·드롭 발생 시:
  - **폴백 1:** 라즈베리파이 **Pico** (~$5, RP2040 ADC 500kHz, 안정적)
  - **폴백 2:** 외장 ADC(ADS 계열) 또는 저가 USB 오실로스코프(~$50)
- ADMP401 **사지 말 것** — 공식 응답 ≤15kHz로 40kHz에서 거의 무응답. 피에조 수신기(MA40S4R)가 40kHz 정합.
- 8Ω 오디오앰프(MAX98357A 등)로 피에조(~2kΩ) 구동 금지 — 임피던스 불일치. 로직버퍼/트랜지스터 사용.

## 5. (선택) 무펌웨어 편의 경로 — ADALM2000

코딩 0·즉시 실험을 최우선하면 [ADALM2000](https://www.digikey.com/en/products/detail/analog-devices-inc/ADALM2000/7019661)(~$287, 신호생성+2ch캡처, libm2k 파이썬). **단 보유 ESP32-S3로 충분하므로 비용상 비추천.**

## 6. 구매 순서

1. **1단계(즉시):** 트랜스듀서 페어(TCT40-16) + OPA344 + 패시브 + 레진 → ~$18. G0(시뮬)은 부품 없이 병행.
2. **2단계(G1 후):** ESP32 ADC가 부족하면 Pico(~$5) 추가.
