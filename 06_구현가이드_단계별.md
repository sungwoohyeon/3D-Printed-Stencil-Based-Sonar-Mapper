## G0 — 시뮬레이션 ✅ (완료)

**목표:** 하드웨어 전, 복원 알고리즘·접근법을 모델 불일치 하에서 검증.

```bash
cd sim
python design_stencil.py        # 스텐실 최적화 → stencil_design.npz
python validate_mismatch.py     # G0 게이트
```
**통과 기준:** 작은 불일치(튜브 보정)에서 타깃검출 recall ≥80%. → **달성**(단일·이중 100%).
**핵심 교훈:** 큰 불일치에서 붕괴(47~53%) → ∴ 실측 H 필수. 자세히 [03](03_시뮬레이션_설계.md).

## G1 — 전자 검증 ⏳ (부품 후)

**목표:** ESP32로 40kHz 송수신, 누설·링다운·단일반사 확인.

1. 회로 조립([05 §1](05_벤치_및_실측H_가이드.md)) — 스피커+버퍼, 마이크+프리앰프+바이어스, **배플**.
2. `spidr_bench.ino` 업로드([05 §2](05_벤치_및_실측H_가이드.md)).
3. 빈 상태 캡처(배경) + 6/10/15cm 판 반사 캡처.

**통과 기준:** 배경 차감 후 6·10·15cm 에코가 **시간축 다른 위치**에 분리되어 보임.
**막히면:** 200kHz ADC 검증 실패 → `SAMPLE_HZ`↓ 또는 Pico. 에코 가림 → 배플·게이트.

---

## G2 — 스텐실 + 1D 실측 H ⏳

**목표:** 스텐실 장착 후, 점반사체 1D 스캔으로 H 측정 → 위치 분해.

1. 스텐실 출력([04](04_스텐실_3D설계가이드.md)) — `stencil_5cm.stl`(×1000!) 또는 `models/SPiDR_stencil_parametric.stl`.
2. 스피커에 스텐실 씌우고 캐비티 밀봉(누설 방지).
3. 1D 실측 H:
   ```bash
   cd bench
   python measure_H.py --port COM5 --pitch 2 --depth 10   # 11점, 손 지그
   ```
4. 단일 타깃 위치 맞히기 → **두 물체 2cm 분해**.

**통과 기준:** 측정 H로 두 반사체를 좌우 분해. H 결맞음·조건수가 합리적(낮음).

---

## G3 — 거친 2D ⏳

**목표:** 실측 H로 2cm 격자 2D 점유맵.

1. 깊이 여러 층(예: 6·8·10·12cm) 각각 1D H 측정 → 2D H 결합. (또는 2D 격자 직접 스캔)
2. 실제 장면(물체 1~3개) 캡처 → `reconstruct_nnls(H, y)` → 점유맵 시각화.
3. 점진적으로 1cm·2cm갭 도전(stretch).

**통과 기준:** 단순 장면의 2D 점유맵 생성. (논문급 1cm/SSIM>80%는 stretch — [00 기대치](00_프로젝트개요.md))
